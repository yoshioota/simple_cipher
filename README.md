# SimpleCipher

OpenSSLのCipherをラップしたライブラリです。
加えて、keyをキャッシュしてあるので実用に耐えられる速さになっています。

## Installation

Add this line to your application's Gemfile:

    gem 'simple_cipher'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simple_cipher
